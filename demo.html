Helium-Modal

Helium-modal is a jQuery modal window script, developed by Harun Eggleton under the open source MIT licence.

Features:

  • No required set widths
  • Fully CSS stylable
  • Recenters if size of modal changes
  • Customizable show/hide animation
  • Triggered through a link or javascript
